faq

Common questions

Is this safe to play? Am I actually hacking something?
No real systems are involved. The server, database, and shell are all simulated. You can't break anything or affect anyone else.
Do I need any prior hacking or security knowledge?
No. Level 1 assumes you know how to use a terminal. If you've written any backend code, you'll be fine. Hints are available whenever you get stuck.
How is this different from TryHackMe or HackTheBox?
Those platforms are broader — full Linux machines, network challenges, CTFs. HackLab is laser-focused on web application vulnerabilities and built for developers, not security specialists. It takes an hour, not weeks.
Will my progress be saved if I close the tab?
Yes — progress is stored locally. As long as you use the same browser, your stage completion and terminal history persist across sessions and server restarts.
Can I use this to teach a class or team?
Absolutely. Each player gets an isolated session. Just share the link — no accounts or setup needed.
